Represents a process concept at a specific event in time.

## Scope

This data type is intended to represent an instance of a temporal event within a defined process.

Loosely taking inspiration from [HL7 Timing](https://www.hl7.org/fhir/datatypes.html#timing).

## Structure

| Name | Cardinality | Type | Description & Constraints |
| --- | --- | --- | --- |
| event | 1..1 | [Event](../data/event) | An event concept. This attribute should solely describe the raw known factual information a specific temporal event. |
| process | 1..1 | [Process](../data/process) | An process tracking concept. This attribute should represent the core concept of a defined process. |
